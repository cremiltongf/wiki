Configuração do meu Fedora Linux 42 (Workstation Edition).

Gerenciador de Extensões

Instalação


sudo flatpak install flathub com.mattjakeman.ExtensionManager

Ajustes do Gnome

Instalação


sudo dnf install gnome-tweaks
Instale as extensões através do Gerenciador de Extensões.

AppIndicator and KStatusNotifierItem Support

Exibe ícones indicadores no painel (uso para o Dropbox). Não é necessário nenhuma configuração, mas é possível mudar o local de exibição por exemplo, que por padrão é a direita.

Mais detalhes: AppIndicator and KStatusNotifierItem Support.

Blur my Shell

Adiciona uma aparência desfocada a diferentes partes do GNOME Shell, incluindo o painel superior, o painel e a visão geral.

Compiz alike magic lamp effect

Adiciona efeito de lâmpada mágica às janelas.

Compiz windows effect

Adiciona efeito de janelas instáveis.

Dash to Dock

Um dock para o Gnome Shell.

Search Light

Abre a pesquisa de aplicativos da visão geral.

Tiling Shell

Adiciona o gerenciamento avançado de janelas.

Ícones

Kora é um tema de ícones SVG para sistemas operacionais GNU/Linux.

Wallpapers

Papéis de parede.

Git

Instalação


sudo dnf install git

Gerenciador de máquinas virtuais

Há um ótimo material no site SysGuides para instalação e configuração.

Docker

Siga o guia do Docker Docs que é sucesso.

Utiliza o AppIndicator and KStatusNotifierItem Support para exibir as notificações.

Visual Studio Code

Façao download do Visual Studio Code.

Instalação


sudo dnf install "code-1.101.1-1750254783.el8.x86_64.rpm"

PowerIso

Instalação


sudo flatpak install flathub com.poweriso.PowerISO

Dropbox

Faça o download do Dropbox.

Instalação


sudo dnf install "nautilus-dropbox-2025.05.20-1.fc42.x86_64.rpm"
Utiliza o AppIndicator and KStatusNotifierItem Support para exibir as notificações.

qBittorrent

Instalação


sudo flatpak install flathub org.qbittorrent.qBittorrent
Utiliza o AppIndicator and KStatusNotifierItem Support para exibir as notificações.

Atalhos personalizados

  1. Em Configurações → Teclado → Atalhos de teclado → Veja e personalize atalhos → Atalhos personalizados
    1. Ctrl + Alt + T → ptyxis → para abrir o Terminal
    2. Ctrl + Alt + E → nautilus → para abrir o Nautilus

Webcam


sudo dnf install v4l-utils
Isso funcionou muito bem para minha webcam usb.

Referência: Fedora Webcam Troubleshooting Guide
