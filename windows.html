Criação de Imagem

Setup

  • Armazenamento no modo AHCI;
  • Habilitar a virtualização;
  • Desativar Fast Boot;
  • Desativar Secure Boot;
  • Ativar PXE no modo Legacy.
Sempre que possível, instale o Windows no modo UEFI e desative o IPv6.

Usuário administrador

Após a instalação do SO, crie o usuário administrador IFPR e defina a senha padrão disponível planilha.

Usuário aluno

Crie o usuário para o estudante chamado aluno e defina a senha padrão:

                    
                    Usuário: aluno
                    Senha: aluno123
Nos LABs do IFPR, o usuário é criado pelo Active Directory.

Configurações

Utilize armazenamento no modo AHCI, habilite suporte a virtualização, desative Fast Boot/Secure Boot e ative o PXE no modo legado.

Otimize a inicialização

Senhas

Coloque um txt na área de trabalho com as senhas, caso alguma aplicação necessite.

Reparo automático

Desativar no Windows 7

                    
                    bcdedit /set {default} bootstatuspolicy ignoreallfailures

Desativar no Windows 10

                    
                    bcdedit /set recoveryenabled NO
Configuração para evitar o reparo automático do SO.

Apache NetBeans

Faça o download do NetBeans.

Esse programa precisa do Java JDK.

Silent Install

                    
                    "Apache-NetBeans-12.5-bin-windows-x64.exe" --silent --state "state.xml"

Baixe o state.xml

Necessário gerar um novo state.xml a cada nova versão do NetBeans que será instalado.

Silent Uninstall

                    
                        "%ProgramFiles(x86)%\NetBeans 8.2\uninstall.exe" --silent
                        "%ProgramFiles%\NetBeans 8.2\uninstall.exe" --silent
                        rd "C:\Users\aluno\AppData\Roaming\NetBeans" /s /q

Em caso de alteração do Java JDK, alterar a configuração:

                    
                    "C:\Program Files\NetBeans 8.2\etc\netbeans.conf"

Diretório de configuração do usuário Aluno:

                    
                    "C:\Users\aluno\AppData\Roaming\NetBeans"
