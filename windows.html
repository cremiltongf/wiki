Reparo automático All lab

Desativando:

                    
                    #windows7
                    bcdedit /set {default} bootstatuspolicy ignoreallfailures

                    #windows10
                    bcdedit /set recoveryenabled NO

Ativando:

                    
                    #windows7
                    bcdedit /set {default} bootstatuspolicy displayallfailures

                    #windows10
                    bcdedit /set recoveryenabled YES

Isso evita o reparo automático do SO em caso de perda de energia. O que pode quebrar o congelamento do SO.

                    REG ADD "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\OptimalLayout" /V EnableAutoLayout /T REG_DWORD /D 0 /F

Variáveis do Sistema All lab

Crie as variáveis para a última versão do Java JDK instalado. No meu caso, possuo duas instalações, o JDK7 por causa do IReport, e o JDK 8 para os programas que necessitam de uma versão mais recente.

Variáveis via Propriedades do Sistema

Acesse as Propriedades do Sistema para configurar as variáveis. Neste exemplo, estou configurando o JDK 1.8.0_191:

                    
                    JAVA_HOME
                    C:\Program Files\Java\jdk1.8.0_191
                    
                    CLASSPATH
                    .;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\htmlconverter.jar;%JAVA_HOME%\jre\lib;%JAVA_HOME%\jre\lib\rt.jar

A variável Path já existe no sistema, então apenas adicione o valor abaixo ao final:

                    
                    Path
                    ;%JAVA_HOME%\bin

No Windows 10:

                    
                    Path
                    %JAVA_HOME%\bin

Variáveis via cmd

Precisa ser administrador. 
                    
                    setx JAVA_HOME "C:\Program Files\Java\jdk1.8.0_191" /m
                    setx CLASSPATH "%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\htmlconverter.jar;%JAVA_HOME%\jre\lib;%JAVA_HOME%\jre\lib\rt.jar" /m
                    setx Path "%Path%;%JAVA_HOME%\bin" /m
