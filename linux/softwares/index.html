Firefox

Instalação


sudo flatpak install flathub org.mozilla.firefox

Configurando políticas


# get flatpak architecture
FLATPAK_ARCH=$(flatpak --default-arch)

# create the user-specific flatpak policies directory
mkdir -p "$HOME/.local/share/flatpak/extension/org.mozilla.firefox.systemconfig/$FLATPAK_ARCH/stable/policies"

# set json file
cp /home/padawan/Downloads/policies.json "$HOME/.local/share/flatpak/extension/org.mozilla.firefox.systemconfig/$FLATPAK_ARCH/stable/policies"

Conteúdo de policies.json 


{
   "policies": {
    "ExtensionSettings": {
      "*": {
        "blocked_install_message": "Instalação bloqueada por políticas de segurança.",
        "installation_mode": "blocked"
      }
    },
    "AppAutoUpdate": false,
    "BackgroundAppUpdate": false,
    "DisableAppUpdate": false,
    "DisableDefaultBrowserAgent": true,
    "DontCheckDefaultBrowser": true,
    "SanitizeOnShutdown": true,
    "BlockAboutAddons": true
  }
}

Referência: Como personalizar o Firefox usando policies.json e policy-templates.

Wine

Instalação


# Download and add the repository key
sudo mkdir -pm755 /etc/apt/keyrings
wget -O - https://dl.winehq.org/wine-builds/winehq.key | sudo gpg --dearmor -o /etc/apt/keyrings/winehq-archive.key -

# Noble
sudo dpkg --add-architecture i386
sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/ubuntu/dists/noble/winehq-noble.sources

# Update
sudo apt update -y

#Install
sudo apt install --install-recommends winehq-stable

# Reboot
reboot
Configuração específica para o Zorin OS 18.1.

Mais informações: Wine Wiki.
