Plano de fundo

Baixe o background.png para a pasta Imagens e defina como plano de fundo.

Configurações

Energia

Economia de energia

  1. Apagar a tela → Nunca
  2. Suspensão automática → Desligada

Personalização

Ferramentas que serão utilizados para personalização.


# install git                        
sudo apt install git

# install gnome-tweaks
sudo apt install gnome-tweaks

# install extension-manager
sudo flatpak install flathub com.mattjakeman.ExtensionManager

Instalando ícones


git clone https://github.com/yeyushengfan258/Win11-icon-theme && cd Win11-icon-theme && ./install.sh

Instalando fonte


git clone https://github.com/mrbvrz/segoe-ui-linux && cd Win11-icon-theme && ./install.sh && fc-cache -f -v

Ajustes

Abra o Gnome-Tweaks:

  1. Na aba Fontes selecione Segoe UI Light;
  2. Na aba Aparência selecione Win11.

Extensões

Instale as extensões abaixo pelo Gerenciador de Extensões.

Desative Zorin Menu e Zorin Taskbar.

ArcMenu

Menu de aplicativos para o GNOME Shell.

Baixe o icon-menu-win11.png (ícone do menu) para a pasta Imagens e o arquivo de configuração (config-arc-menu) para restaurar no ArcMenu.

Dash to Panel

Uma barra de tarefas com ícones para o Gnome Shell.

Baixe o arquivo de configuração (config-dash-to-panel) para restaurar no Dash to Panel.

Organização da barra

Date Menu Formatter

Permite personalizar a exibição da data no painel.

Configuração: 


# Align Text
Right

# Pattern
HH:mm\ndd/MM/yyyy

Disable extension updates

Desativa as atualizações automáticas de extensões no GNOME.

Zorin Appearance

Área de Trabalho

ícones

  1. Ícones da área de trabalho → Ativo
  2. Tamanho do ícone → Pequeno
  3. Deixar ativo → Somente a Lixeira

Animações

Desative as animações da interface:


gsettings set org.gnome.desktop.interface enable-animations false
