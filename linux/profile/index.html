Configurações para o perfil do usuário Aluno.

Aparência

Efeitos

  1. Efeitos da área de trabalho e janela → Desabilitado.

Planos de Fundo

  1. Imagens → Baixe o arquivo background.png para a pasta ~/.imagens do Aluno e selecione-o.

Temas

  1. Cursor do Mouse → Baixe e extraia Windows-cursors para ~/.icons do Aluno e selecione-o.
  2. Aplicativos → Baixe e extraia o tema Windows-10 para ~/.themes do Aluno e selecione-o.
  3. Ícones → Baixe e extraia Windows 10 icon theme para ~/.icons do Aluno e selecione-o.
  4. Área de Trabalho → Selecione cinnamon.

Preferências

Applets

  1. Na aba Download → instale o CinnVIIStarkMenu.
Substitua o menu padrão do Linux Mint pelo CinnVIIStarkMenu e faça as devidas configurações.

Área de Trabalho

  1. Ícones da Área de Trabalho → Lixeira → Habilitado.

Notificações

  1. Habilitar notificações → Desabilitado.

Proteção de Tela

  1. Atraso antes de iniciar a proteção de tela → Nunca.
  2. Bloquear o computador quando entrar em suspensão → Desabilitado.
  3. Bloquear o computador após a proteção de tela iniciar → Desabilitado.

Hardware

Gerenciamento de energia

  1. Desligar a tela quando inativo por → Nunca.
  2. Suspender quando inativo por → Nunca.

Gerenciador de Atualizações

Aplica-se ao usuário IFPR e Aluno.
  1. Editar → Preferências → Desabilitar tudo.
