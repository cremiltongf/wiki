Padrão de nome de computadores e Ativação de serviços para manter a integridade do SO.

Alterar Hostname

sudo vim /etc/hostname
sudo vim /etc/hosts

Renomeie o computador conforme dados abaixo:

LAB Nome Grupo
LAB01 lab1m01 WORKGROUP
LAB02 lab2m01 WORKGROUP
LAB03 lab3m01 WORKGROUP
LAB04 lab4m01 WORKGROUP
LAB05 lab5m01 WORKGROUP

Restore Profile

Habilitando o serviço

sudo systemctl enable reset-user.service

Desabilitando o serviço

sudo systemctl disable reset-user.service
