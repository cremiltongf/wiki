Configuração padrão do Sistema Operacional Linux Mint 22.1.

Partição do SO

Utilize todo espaço da unidade para instalação da distribuição.

Usuário Root

Criar usuário root IFPR e defina a senha padrão disponível na planilha→.

Usuário Padrão

Crie o usuário Aluno e configure o login automático.

Usuário: aluno
Senha: aluno123

Login Automático

  1. Janela de Início de Sessão ↔ Aba Usuários ↔ configure o usuário Aluno:
    1. Nome do usuário: aluno
    2. Atraso antes da conexão (em segundos): 0

Linux Update

Instale todas as atualizações disponíveis.

sudo apt-get update -y && sudo apt-get upgrade -y && sudo apt-get dist-upgrade -y
