Ferramentas de apoio

O veyon auxilia no controle de pcs pela rede. O Toolwiz Time Freeze ajuda a manter o Sistema Operacional sempre intacto.

Veyon

O Veyon→ permite que o professor monitore o laboratório, além de fornecer ferramentas para o auxíliar os alunos remotamente. Além disso, podemos executar scripts/bat nos computadores, o que o torna uma ótima ferramenta de apoio para TI.

Executando comandos nos computadores

O comando abaixo abre o prompt no pc cliente:

cmd /k

Mais detalhes sobre cmd /k →.

Um exemplo de execução de .bat:

cmd /k "C:\Users\aluno\Desktop\software\tools.bat"

Atualizando GPO:

cmd /k "gpupdate /force"

Toolwiz Time Freeze

É o software responsável por congelar o Sistema Operacional Windows a fim mantê-lo intacto.

Descongelando o SO

cmd /k "C:\Program Files\Toolwiz Time Freeze 2017\ToolwizTimeFreeze.exe" /unfreeze /usepass=SENHA

Congelando o SO

cmd /k "C:\Program Files\Toolwiz Time Freeze 2017\ToolwizTimeFreeze.exe" /freezealways /usepass=SENHA

Checar o status

cmd /k "C:\Program Files\Toolwiz Time Freeze 2017\ToolwizTimeFreeze.exe" /check /usepass=SENHA
