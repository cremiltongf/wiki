Ajuste final

Ajustes de otimização para que seja possível tornar o SO mais performático na inicialização. Também temos orientações para tornar programas como o VLC Media Player padrão do Windows.

Aplicativos e recursos

Aplicativos padrão

  1. Definir aplicativo padrão
Defina o VLC Media Player e o Google Chrome como aplicativos padrões.

Inicialização

  1. Desativar inicializações desnecessárias

Otimize a inicialização

Acesse msconfig como Administrador.

Proteção do Sistema

Desative a Proteção do Sistema através das Configurações Avançadas do Sistema:

  1. Configurações ↔ Sistema ↔ Sobre ↔ Configurações Avançadas do Sistema ↔ Proteção do Sistema ↔ Configurar... ↔ Desativar a proteção do sistema.

Agendador de Tarefas

Desative as tarefas encontradas em:

  1. Tarefas da Biblioteca do Agendador ↔ Desativar tudo
  2. Tarefas de Microsoft ↔ Windows ↔ AppListBackup ↔ Desativar Tudo
  3. Tarefas de Microsoft ↔ Windows ↔ License Manager ↔ Desativar TempSignedLicenseExchange

Histórico de Navegação

Remover histórico do Edge, Chrome e Firefox.

Configuração de proxy

Possível definir pelo Active Directory. Configure para o usuário IFPR e Aluno.

Dados do proxy:

192.168.0.6:3128

Opções de pastas

Configure para o usuário IFPR e Aluno.
