Configuração do clone

Nome de computadores, ativação dos softwares e demais ajustes necessários após a clonagem para o bom funcionamento do computador no ambiente do laboratório.

Nome de computadores

Renomei o computador conforme orientações← para cada laboratório.

Ativar AutoCAD

Execute o AutoCAD (necessário internet) e siga os passos que aparecer na tela e, informe a chave de ativação→ caso solicitado.

Ativar Revit

Execute o Revit (necessário internet) e siga os passos que aparecer na tela e, informe a chave de ativação→ caso solicitado.

Habilitar Toolwiz Time Freeze

É o software responsável por congelar o Sistema Operacional Windows a fim mantê-lo intacto.

Descongelando o SO

cmd /k "C:\Program Files\Toolwiz Time Freeze 2017\ToolwizTimeFreeze.exe" /unfreeze /usepass=SENHA

Congelando o SO

cmd /k "C:\Program Files\Toolwiz Time Freeze 2017\ToolwizTimeFreeze.exe" /freezealways /usepass=SENHA

Checar o status

cmd /k "C:\Program Files\Toolwiz Time Freeze 2017\ToolwizTimeFreeze.exe" /check /usepass=SENHA

Habilitar Reboot Restore RX

É o software responsável por congelar o Sistema Operacional Windows a fim mantê-lo intacto.
