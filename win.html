Criação de Imagem

Configuração inicial para o Hardware e Sistema Operacional Windows.

Configurações

  • Armazenamento no modo AHCI;
  • Habilitar virtualização;
  • Desativar Fast Boot;
  • Desativar Secure Boot;
  • Ativar PXE no modo Legacy ou UEFI;
  • Ativar Wake On LAN (WOL).

Modo UEFI

Quando possível instalar o Windows no modo UEFI.

Usuário Administrador

Criar usuário Administrador IFPR e definir senha padrão disponível na planilha ↗.

Realize as configurações para o SO com a conta de Administrador local ou de rede, a fim de evitar múltiplas autenticações.

Configuração de driver para ativação WOL

  1. Abra o Gerenciador de Dispositivos;
  2. Acesse Adaptadores de rede e abra a categoria;
  3. Clique com o botão direito no LAN Controller e selecione Propriedades;
  4. Abra a guia Gerenciamento de energia;
  5. Marque as caixas:
    1. Permitir que este dispositivo acorde o computador;
    2. Só permita que um pacote mágico acorde o computador.
  6. Agora, abra a guia Avançado e escolha Energy Efficient Ethernet e Defina essa opção como Desativado;
  7. Confirme as alterações clicando em OK.

Partições do SO

Defina as partições do computador conforme tabela abaixo:

Descrição Capacidade
Nuvem01 Mínimo de 10GB
Windows Restante da capacidade
Caso o equipamento possua unidades extras de armazenamento, utilize todo espaço da unidade para partição nuvem01 e/ou nuvem02.

Windows Update

Instale todas as atualizações do Windows.

Atualização de drivers

Use softwares como EasyDriver ↗, Driver Booster ↗ ou DriverMax ↗ para realizar essa tarefa.

Fast Boot

Execute no prompt como Administrador:

powercfg.exe /hibernate off
Ajuste necessário para o desligamento completo do SO. Aplicável ao Windows 8/10/11.

Propriedades do Sistema

Aba Proteção do Sistema

  1. Configurar → Desativar a proteção do sistema.

Reparo automático

Desativar no Windows 7

Execute o comando abaixo pelo prompt como Administrador:

bcdedit /set {default} bootstatuspolicy ignoreallfailures

Desativar no Windows 10

Execute o comando abaixo pelo prompt como Administrador:

bcdedit /set recoveryenabled NO
Ajuste necessário para evitar o reparo automático do SO, o que pode "quebrar" o Windows.

Sistema

Notificações e ações

  1. Desativar → Tudo.

Energia e suspensão

  1. Tela → Quando conectado, desligar após → Nunca;
  2. Suspender → Quando conectado, o computador será suspenso após → Nunca.

Personalização

Cores

  1. Efeitos de transparência → Desativado.

Tela de bloqueio

  1. Tela de Fundo → Imagem;
  2. Baixe o papel da tela de bloqueio ↗ para a pasta imagem e defina para a tela;
  3. Receba dados interessantes... → Desativado.

Iniciar

  1. Mostrar lista de aplicativos no menu Iniciar → Ativado;
  2. Todas as outras opções → Desativadas;
  3. Escolher quais pastas são exibidas em Iniciar:
    1. Configurações → Ativado;
    2. Pasta pessoal → Ativado;
    3. Todas as outras opções → Desativadas.

Barra de Tarefas

  1. Combinar botões da barra de tarefas → Quando a barra de tarefas estiver cheia.
