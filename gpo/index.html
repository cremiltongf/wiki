Configurações do Windows

  • Scripts (Inicialização ou Encerramento)
    1. Em Inicialização adicione wallpaper.bat:
      2. @echo off

setlocal

set "IMAGE=background.png"
set "SERVER=\\192.168.x.xx\datapublic\0\Background\%IMAGE%"
set "DIRIMAGE=C:\Background"

if not exist "%DIRIMAGE%\%IMAGE%" (
  md "%DIRIMAGE%"
  copy /y "%SERVER%" "%DIRIMAGE%"
)

exit
  • Configurações de Segurança
    1. Políticas de Conta → Políticas do Kerberos
      1. Tolerância máxima para a... → 60
    2. Políticas Locais → Opções de Segurança
      1. Contas: bloquear contas da Microsoft → Os usuários não podem adicionar ou fazer...
      2. Membro de domínio: duração máxima de senha... → 30
    3. Serviços do sistema
      1. Backup do Windows → Desabilitado
      2. Otimizador de unidade → Manual
      3. SysMain → Desabilitado
      4. Windows Search → Desabilitado
      5. Microsoft Edge Elevation Service → Desabilitado
      6. Microsoft Edge Update Service (edgeupdate) → Desabilitado
      7. Microsoft Edge Update Service (edgeupdatem) → Desabilitado
