Ativar o Windows

Todas as máquinas possuem licença OEM.

Para acessar a key pelo prompt:


powershell "(Get-WmiObject -query 'select * from SoftwareLicensingService').OA3xOriginalProductKey"

Referência: How to find your Windows Product Key using the Command Prompt or Powershell

Habilitar Deep Freeze Cloud

É o software responsável por congelar o Sistema Operacional Windows a fim mantê-lo intacto.

Descongelando o SO


cmd /k "C:\Windows\SysWOW64\DFC.exe" PWD /BOOTTHAWED

Congelando o SO


cmd /k "C:\Windows\SysWOW64\DFC.exe" PWD /FREEZENEXTBOOT

Checar o status


cmd /k "C:\Windows\SysWOW64\DFC.exe" get /ISFROZEN

Habilitar Toolwiz Time Freeze

É o software responsável por congelar o Sistema Operacional Windows a fim mantê-lo intacto.

Descongelando o SO


cmd /k "C:\Program Files\Toolwiz Time Freeze 2016\ToolwizTimeFreeze.exe" /unfreeze /usepass=PWD

Congelando o SO


cmd /k "C:\Program Files\Toolwiz Time Freeze 2016\ToolwizTimeFreeze.exe" /freezealways /usepass=PWD
