Habilitar Deep Freeze Cloud

Descongelando o SO


cmd /k "C:\Windows\SysWOW64\DFC.exe" PWD /BOOTTHAWED

Congelando o SO


cmd /k "C:\Windows\SysWOW64\DFC.exe" PWD /FREEZENEXTBOOT

Checar o status


cmd /k "C:\Windows\SysWOW64\DFC.exe" get /ISFROZEN
