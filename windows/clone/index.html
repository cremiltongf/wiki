Configuração do SO clonado

Ingresso no domínio, ativação dos softwares da Autodesk e demais ajustes necessários após a clonagem para o bom funcionamento do computador no ambiente do laboratório.

Computador no domínio

Adicione o computador no domínio conforme orientações → para cada laboratório.

Atualizar as GPOs

Atualize as GPOs para que todas as diretivas sejam atualizadas e aplicadas.

gpupdate /force

Ativar AutoCAD

Execute o AutoCAD (necessário internet) e siga os passos que aparecer na tela e, informe a chave de ativação ↗ caso solicitado.

Ativar Revit

Execute o Revit (necessário internet) e siga os passos que aparecer na tela e, informe a chave de ativação ↗ caso solicitado.

Habilitar Toolwiz Time Freeze

Após todas as configurações, reinicie o computador antes de ativar o congelamento.

Por último, ative o Toolwiz Time Freeze →.
