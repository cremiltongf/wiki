Inicialização

Otimize a inicialização

Acesse msconfig como Administrador.

Proteção do Sistema

Desative a Proteção do Sistema através das Configurações Avançadas do Sistema:

  1. Configurações → Sistema → Sobre → Configurações Avançadas do Sistema → Proteção do Sistema → Configurar... → Desativar a proteção do sistema.

Agendador de Tarefas

Desative as tarefas encontradas em:

  1. Tarefas da Biblioteca do Agendador → Desativar tudo
  2. Tarefas de Microsoft → Windows → AppListBackup → Desativar Tudo
  3. Tarefas de Microsoft → Windows → License Manager → Desativar TempSignedLicenseExchange
  4. Tarefas de Microsoft → Windows → Windows Defender → Desativar Tudo

Histórico de Navegação

Remover histórico do Edge, Chrome e Firefox.

Generalizar Instalação do Windows

Antes de criar a imagem do Windows, é necessário generalizar a instalação atual. No meu cenário, estou usando um xml para automatizar as configurações iniciais após a clonagem da imagem generalizada. 

sysprep.exe /generalize /oobe /unattend:C:\MYCONF.xml /shutdown

Referências: Microsoft-Windows-Shell-Setup e Sysprep

Erro no Sysprep

Se ocorrerem erros, examine o arquivo %WINDIR%\system32\Sysprep\panther\setupact.log e remova o App que estiver causando a falha através do PowerShell: 


Get-AppxPackage -AllUsers | Where-Object {$_.PackageFullName -eq "NomeDoPackage"} | Remove-AppxPackage -AllUsers

Referência: Get-AppxPackage
