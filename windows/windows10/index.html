Ajustes para Windows 10

Ajustes necessários para o Windows 10 e 11. A fim de tornar um pouco mais performático o ambiente configurado no camputador.

Sistema

Notificações e ações

  1. Desativar → Tudo.

Energia e suspensão

  1. Tela → Quando conectado, desligar após → Nunca;
  2. Suspender → Quando conectado, o computador será suspenso após → Nunca.

Personalização

Cores

  1. Efeitos de transparência → Desativado.

Tela de bloqueio

  1. Tela de Fundo → Imagem;
  2. Baixe o papel da tela de bloqueio↗ para a pasta imagem e defina para a tela;
  3. Receba dados interessantes... → Desativado.

Iniciar

  1. Mostrar lista de aplicativos no menu Iniciar → Ativado;
  2. Todas as outras opções → Desativadas;
  3. Escolher quais pastas são exibidas em Iniciar:
    1. Configurações → Ativado;
    2. Pasta pessoal → Ativado;
    3. Todas as outras opções → Desativadas.

Barra de Tarefas

  1. Combinar botões da barra de tarefas → Quando a barra de tarefas estiver cheia.

Jogos

Xbox Game Bar

  1. Habilite a Xbox Game Bar... → Desativado.

Modo de Jogo

  1. Modo de Jogo → Desativado.

Privacidade

Geral

  1. Permitir que o Windows controle a abertura de aplicativos... → Ativado;
  2. Desativar as demais opções.

Histórico de atividades

  1. Tudo Desativado.

Aplicativos em segundo plano

  1. Permitir a execução de aplicativos em segundo plano → Desativado.

Opções de pastas

Configurações do Edge

Configurar a página inicial do navegador Edge.
