Softwares

Instalação de softwares, links para download, configurações e ajustes necessários para o bom funcionamento do ambiente.

Execute os softwares ao menos uma vez na conta do usuário Aluno para que sejam criadas as configurações de ambiente.

Silent Install

"7z1805-x64.exe" /S

Anaconda

Instale este software apenas para o usuário Aluno.

Faça o download do Anaconda→.

Configuração de proxy

Edite o arquivo c:\users\aluno\.condarc e adicione:

show_channel_urls: true
allow_other_channels: true

proxy_servers:
    http: 192.168.0.6:3128
    https: 192.168.0.6:3128

ssl_verify: false

Referência: Update the .condarc file→.

Instalando o Docker

Faça o download e instale o Docker Desktop→.

Silent Install

start /w "" "Docker Desktop Installer.exe" install

Permissão para criar container

Adicione o usuário Aluno ao grupo docker-users. Execute o comando no prompt como Administrador:

net localgroup docker-users aluno /add

Configurando proxy

Edite o arquivo C:\Users\aluno\.docker\config.json:

"proxies": {
    "default": {
        "httpProxy": "http://192.168.0.6:3128",
        "httpsProxy": "https://192.168.0.6:3128",
        "noProxy": "127.0.0.0/8,localhost"
    }
},

Configurações

  1. General
    1. Start Docker Desktop when you sign in to your computer ↔ Desativado
    2. Open Docker Dashboard when Docker Desktop starts ↔ Desativado
    3. Send usage statistics ↔ Desativado
  2. Software updates
    1. Automatically check for updates ↔ Desativado

Imagens a serem baixados:

  1. alpine ↔ docker pull alpine
  2. node ↔ docker pull node
  3. nginx ↔ docker pull nginx
  4. mysql ↔ docker pull mysql

Proxy para o IntelliJ

Proxy para o Maven

Crie o arquivo settings.xml em C:\Users\aluno\.m2 com o seguinte conteúdo:

<?xml version="1.0" encoding="UTF-8"?>
<settings xmlns="http://maven.apache.org/SETTINGS/1.0.0"
    xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:schemaLocation="http://maven.apache.org/SETTINGS/1.0.0 http://maven.apache.org/xsd/settings-1.0.0.xsd">
    <proxies>
        <!-- Proxy for HTTP -->
        <proxy>
            <id>proxyifpr</id>
            <active>true</active>
            <protocol>http</protocol>
            <username></username>
            <password></password>
            <host>192.168.0.6</host>
            <port>3128</port>
            <nonProxyHosts>localhost</nonProxyHosts>
        </proxy>

        <!-- Proxy for HTTPS -->
        <proxy>
            <id>proxyifpr</id>
            <active>true</active>
            <protocol>https</protocol>
            <username></username>
            <password></password>
            <host>192.168.0.6</host>
            <port>3128</port>
            <nonProxyHosts>localhost</nonProxyHosts>
        </proxy>
    </proxies>

</settings>
