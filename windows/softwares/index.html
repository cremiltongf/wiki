Softwares

Instalação de softwares, links para download, configurações e ajustes necessários para o bom funcionamento do ambiente.

Execute os softwares ao menos uma vez na conta do usuário Aluno, para que sejam criadas as configurações de ambiente. Faça a instalação dos programas com uma conta de Administrador.

7-zip

Faça o download do 7-zip↗.

Silent Install

"7z1805-x64.exe" /S

Android Studio

Faça o download do Android Studio↗.

Executar o programa no perfil do aluno, para terminar de adicionar os recursos extras.

Apache NetBeans

Faça o download do NetBeans↗.

Esse programa precisa do Java JDK.

Silent Install

"Apache-NetBeans-12.5-bin-windows-x64.exe" --silent --state "state.xml"

Baixe o state.xml↗

Necessário gerar um state.xml para cada versão do NetBeans.

Silent Uninstall

"%ProgramFiles(x86)%\NetBeans 8.2\uninstall.exe" --silent
"%ProgramFiles%\NetBeans 8.2\uninstall.exe" --silent
rd "C:\Users\aluno\AppData\Roaming\NetBeans" /s /q

Em caso de alteração do Java JDK, alterar a configuração:

"C:\Program Files\NetBeans 8.2\etc\netbeans.conf"

Diretório de configuração do usuário Aluno:

"C:\Users\aluno\AppData\Roaming\NetBeans"

Bonjour ou mDNS

Bonjour

Faça o download do Bonjour↗.

Silent Install

msiexec /i "Bonjour64.msi" /qb
Instale apenas no Windows 7 Professional. Para o Windows 10, ative mDNS.

mDNS - Resolve .local para Windows 10

Para o Windows 10, ative o recurso mDNS:

reg add "HKLM\Software\Policies\Microsoft\Windows NT\DNSClient" /v EnableMulticast /t REG_DWORD /d 1 /f

Java - Variáveis de ambiente

Acesse as Propriedades do Sistema para configurar as variáveis. No exemplo abaixo a configuração se aplica ao JDK 17.

Para o Windows 7/10 crie a variável JAVA_HOME e adicione o valor:

C:\Program Files\Java\jdk-17.0.2
A variável Path já existe no sistema.

Para Windows 7 em Path

;%JAVA_HOME%\bin

Para Windows 10 em Path:

%JAVA_HOME%\bin
Crie as variáveis para a última versão do Java JDK instalado.
