Login no Active Directory

Orientações para login no Active Directory e demais detalhes das regras aplicadas ao computador que foi ingressado ao domínio.

Papel de Parede

Baixe o papel de parede↗.

Configuração definida para o perfil do aluno pelo Active Directory.

Via script

@echo off
    reg add "HKCU\control panel\desktop" /v wallpaper /t REG_SZ /d "" /f
    reg add "HKCU\control panel\desktop" /v wallpaper /t REG_SZ /d "C:\Users\USERNAME\Pictures\background.png" /f
    reg delete "HKCU\Software\Microsoft\Internet Explorer\Desktop\General" /v WallpaperStyle /f
    reg add "HKCU\control panel\desktop" /v WallpaperStyle /t REG_SZ /d 2 /f
    RUNDLL32.EXE user32.dll,UpdatePerUserSystemParameters
exit

Configuração de proxy

192.168.0.6:3128

Configuração definida pelo Active Directory.
