Configurações para o perfil do usuário Aluno e do usuário IFPR.

Opções de pastas

Senhas

Coloque a lista de senhas na área de trabalho do Aluno em um arquivo Senhas.txt, caso necessário.

Se aplica apenas para o usuário Aluno.

Sistema

Notificações e ações

  1. Desativar ↔ Tudo.

Energia e suspensão

  1. Tela ↔ Quando conectado, desligar após ↔ Nunca;
  2. Suspender ↔ Quando conectado, o computador será suspenso após ↔ Nunca.

Personalização

Tela de fundo

  1. Tela de fundo ↔ Imagem;
  2. Escolher sua imagem ↔ Baixe o arquivo background.png→ e selecione-o.
Se aplica apenas para o usuário Aluno.

Cores

  1. Efeitos de transparência ↔ Desativado.

Tela de bloqueio

  1. Receba dados interessantes... ↔ Desativado.

Jogos

Xbox Game Bar

  1. Habilite a Xbox Game Bar... ↔ Desativado.

Modo de Jogo

  1. Modo de Jogo ↔ Desativado.

Privacidade

Geral

  1. Permitir que o Windows controle a abertura de aplicativos... ↔ Ativado;
  2. Desativar as demais opções.

Histórico de atividades

  1. Tudo Desativado.

Aplicativos em segundo plano

  1. Permitir a execução de aplicativos em segundo plano ↔ Desativado.
