Setup e SO

Configuração padrão para o Hardware novo e criação de uma ISO para instalação do Windows.

Configurações

  • Armazenamento no modo AHCI;
  • Habilitar Virtualização;
  • Desativar Fast Boot;
  • Desativar Secure Boot;
  • Ativar Preboot Execution Environment (PXE);
  • Ativar Wake On LAN (WOL).

ISO Windows

Através do Generate autounattend.xml files for Windows 10/11→ é possível personalizar um xml para automatizar alguns passos do processo de instalação do Windows. Adicione o xml ao iso do Windows usando o AnyBurn→.

Através dessa ferramenta é possível remover os bloatwares (OneDrive, Office 365, Cortana e etc.) da instalação, além de outras configurações. Também é possível pré-configurar usuários para o SO.
