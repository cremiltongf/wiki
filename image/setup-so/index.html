Configuração padrão para o Hardware e configuração de uma iso para instalação automatizada do Windows.

Setup Default

  • Armazenamento no modo AHCI;
  • Habilitar Virtualização;
  • Desativar Fast Boot;
  • Desativar Secure Boot;
  • Ativar Preboot Execution Environment (PXE);
  • Ativar Wake On LAN (WOL).

ISO Windows

Através do Generate autounattend.xml files for Windows 10/11→ é possível criar um xml para automatizar o processo de instalação do Windows. Para a edição da iso, utilize o AnyBurn→.

Através dessa ferramenta é possível remover os bloatwares (OneDrive, Office 365, Cortana e etc.) da instalação, além de outras configurações.
