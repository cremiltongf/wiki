Ativar o Windows

Todas as máquinas possuem licença OEM.

Para ver a key via prompt como Administrador:

wmic path SoftwareLicensingService get OA3xOriginalProductKey

Para ver a key via powerShell como Administrador:

powershell "(Get-WmiObject -query 'select * from SoftwareLicensingService').OA3xOriginalProductKey"

Referência: How to find your Windows Product Key using the Command Prompt or Powershell→

Habilitar Deep Freeze Cloud

É o software responsável por congelar o Sistema Operacional Windows a fim mantê-lo intacto.

Descongelando o SO

cmd /k "C:\Windows\SysWOW64\DFC.exe" password /BOOTTHAWED

Congelando o SO

cmd /k "C:\Windows\SysWOW64\DFC.exe" password /FREEZENEXTBOOT

Checar o status

cmd /k "C:\Windows\SysWOW64\DFC.exe" get /ISFROZEN

Este comando retorna 0 (descongelando) ou retorna 1 (congelado).

Habilitar Toolwiz Time Freeze

É o software responsável por congelar o Sistema Operacional Windows a fim mantê-lo intacto.

Descongelando o SO

cmd /k "C:\Program Files\Toolwiz Time Freeze 2017\ToolwizTimeFreeze.exe" /unfreeze /usepass=password

Congelando o SO

cmd /k "C:\Program Files\Toolwiz Time Freeze 2017\ToolwizTimeFreeze.exe" /freezealways /usepass=password

Checar o status

cmd /k "C:\Program Files\Toolwiz Time Freeze 2017\ToolwizTimeFreeze.exe" /check /usepass=password
